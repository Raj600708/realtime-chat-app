# Real-Time Chat Application

A full-stack real-time chat application built with Node.js, Express.js, MongoDB, Mongoose, Socket.IO, EJS, and Multer.

## Live Demo

https://realtime-chat-app-gpx8.onrender.com

## Features

- Real-time messaging using Socket.IO
- User Authentication (Login, Register, Logout)
- Online/Offline Status
- Last Seen Tracking
- Typing Indicators
- Read Receipts (Seen Ticks)
- Message Reply Feature
- Message Deletion
- Emoji Picker Support
- Image Sharing
- Image Preview Modal
- Image Download Support
- Notification Sounds
- Dark Mode
- Responsive Mobile-Friendly UI

## Tech Stack

## Installation

```bash
git clone <repository-url>
cd realtime-chat-app
npm install
npm start
```
